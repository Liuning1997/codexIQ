# Codex Radar 桌面降智雷达

`radar.py` 是可拖动、始终置顶、可手动关闭的桌面仪表盘。它从 Codex Radar 的公开摘要接口读取数据，并缓存最近一次成功响应，以便短暂离线时仍可显示。

推荐规则：先计算当前全部评测配置的 IQ 中位数，仅保留 IQ 不低于中位数的配置；再选取单位等价 API 费用 IQ 最高的一项作为“当前平衡首选”。同时显示高 IQ 与时间优先的建议。

`watcher.py` 轮询 `Codex.exe`：Codex 启动后显示仪表盘，退出后终止仪表盘。手动关闭雷达时，当前 Codex 会话内不会重新弹出；下一次重新启动 Codex 会恢复。

仪表盘表头的齿轮按钮打开圆角设置侧栏，可拖动调整界面/背景色调、窗口透明度、天体透明度与对比度；界面色和背景色均支持取色器，并同步显示 RGB/HEX。收起按钮只保留表头，显示当前时间、IQ 与推荐模型。IQ 趋势会按共同历史起点对齐，短历史模型仍以当前点保留在图中。

运行一次 `install.ps1` 会在当前 Windows 用户的“启动”文件夹创建 `Codex Radar Watcher.lnk`，使守护程序在登录后自动运行。

数据来自 [Codex 雷达](https://codexradar.com/)。其公开摘要接口要求保留数据来源标注；完整 API 与二次开发权限请遵循站点说明申请授权。
